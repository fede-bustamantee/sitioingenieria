Iniciamos un párrafo para probar etiquetas inline(en linea)
La etiqueta "b" nos permite hacer negrita
La etiqueta "i" nos permite hacer cursiva
La etuqueta "u" nos permite hacer subrrallado
Esto está con los tres efectos de estilo


Las etiquetas "b" "i" y "u" están marcadas como obsoletas, se dice que estan deprecadas
La etiqueta recomendada para marcar como importante a una palabra o frase dentro de nuestro párrafo es "strong"
Esto está escrito con la etiqueta "strong" para marcar su relevancia
La etiqueta recomendada para destacar de otra manera una palabra o frase dentro de nuestro párrafo es "em"
Esto está escrito con la etiqueta "em" para marcar su relevancia de otra manera

La etiqueta "br" nos permite hacer un salto de linea, sin cambiarnos de párrafo
